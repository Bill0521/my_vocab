# MyVocab

A personal vocabulary learning app for Android, built with Flutter.

## Features
- **Flashcard Learning**: Review words with spaced repetition.
- **Offline Only**: All data stored locally in SQLite.
- **CSV Import/Export**: Manage your vocabulary.
- **Pronunciation**: Uses system TTS.

## How to Build & Install
1. **Install Flutter**: Make sure Flutter SDK is installed.
2. **Get Dependencies**: Run `flutter pub get`.
3. **Build APK**: Run `flutter build apk --release --target-platform android-arm64`.
4. **Install**: Copy `build/app/outputs/flutter-apk/app-release.apk` to your phone and install.
